Lista de programas 1.

Programas servidores e clientes criados na linguagem C# usando sockets para comunicação sincrona. 

Sempre que for executar uma questão, deve-se iniciar o servidor.

Caso queira compilar o programa, apenas abra o codigo com o visual studio communit.

Caso não queira compilar e apenas executar o programa criado, execute pelo .exe que está na pasta bin\Debug\net6.0-windows dentro do projeto.
